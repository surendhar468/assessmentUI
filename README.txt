# Retail Assessment

This application contains Web UI design files for searching Customer Name/Policy Number. 

This project contains web design logic written using Javascript, HTML5 and CSS.

## Installation
1. Open the project by importing the Pom into an IDE, e.g. Eclipse, IntelliJ

## Test / Run
1. Open the project by importing the Pom into an IDE, e.g. Eclipse, IntelliJ
2. Go to html files folder inside in src/main/webapp/pages folder.
3. Run the html file using browser.
